Relación entre el voto por el Partido Ganador y el Tipo de Circunscripción (Urbano vs Rural)

¿Qué estamos analizando?

Este análisis investiga si la victoria del partido ganador (PDC o LIBRE) está asociada al tipo de circunscripción (Urbana o Rural). Se aplica una prueba Chi-cuadrado para determinar si las variables son independientes y una prueba T-Student para comparar medias de votos.
